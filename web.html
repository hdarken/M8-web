M8 - Pràctica GIT

Definició

Git és un programari de control de versions dissenyat per Linus Torvalds, pensant en l'eficiència, la confiabilitat i compatibilitat del manteniment de versions d'aplicacions quan aquestes tenen un gran nombre d'arxius de codi font.

Propòsit

El seu propòsit és portar registre dels canvis en arxius de computadora incloent coordinar el treball que diverses persones realitzen sobre arxius compartits en un repositori de codi.

Plataforma més comuna

La plataforma més comuna és Github. Aquesta permet allotjar projectes utilitzant el sistema de control de versions Git.
